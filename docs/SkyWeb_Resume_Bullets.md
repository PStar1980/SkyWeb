## One-Line Resume Project Description

Built a full-stack macroeconomic analytics platform using React, ASP.NET Core Web API, C#, PostgreSQL, Dapper, Apache ECharts, D3, and Node.js control-plane services for dashboards, alert rules, signal notifications, and time-series visualization.

## Full-Stack .NET / React Bullets

- Built a full-stack macroeconomic analytics platform with React/Vite frontend, ASP.NET Core Web API backend, PostgreSQL database, Dapper/Npgsql data access, and ECharts/D3 visualization components.
- Migrated SkyWeb from a Node-backed prototype lane to a dedicated ASP.NET Core/C# API through incremental route-family cutover, preserving app stability while expanding .NET capability.
- Implemented authenticated member workflows for profile management, preferences, saved views, custom dashboards, alert rules, and Signal Center notification lifecycle.
- Developed reusable chart architecture with ECharts wrappers, D3 helper utilities, time-series adapters, adaptive axes, rich tooltips, threshold overlays, and optional alert-event markers.
- Designed local development and validation flows with health checks, Swagger/OpenAPI, environment templates, source-only repo zips, and generated repository maps.

## Backend / API Bullets

- Implemented ASP.NET Core Web API endpoints for public macro data, auth/session handling, profile/preferences, saved views, dashboards, alert rules, alert events, and signal notifications.
- Used Dapper and Npgsql to query PostgreSQL macro, auth, and member schemas while preserving frontend response contracts during Node-to-C# migration.
- Built app-scoped authentication with BCrypt password validation, opaque bearer sessions, SHA-256 session-token hashes, and permission-aware protected endpoints.
- Separated analytics/member API ownership from operational execution by keeping alert evaluation, ingestion, workers, and repository utilities in the SkyServer Node.js control plane.
- Added Swagger/OpenAPI and health endpoints to support API inspection, smoke testing, and local validation.

## Data / BI / Analytics Bullets

- Built macroeconomic analytics dashboards over historical time-series data, including curated macro views, single-indicator drilldowns, dashboard cards, and row-level data tables.
- Modeled alert rules and signal notifications around macro indicators and view metrics, connecting rule evaluation to dashboard summaries, Signal Center workflows, and chart overlays.
- Developed reusable visualization components for dense financial/economic time series with adaptive date ranges, compact numeric formatting, hover tooltips, and alert threshold annotations.
- Created a productized macro cockpit for rates, inflation, FX, labor, liquidity, housing, and regime-style analytical lenses.
- Used PostgreSQL-backed views and service-layer APIs to serve curated macro datasets to React dashboard and chart surfaces.

## AI Automation / Platform Bullets

- Maintained structured phase documentation, validation checklists, repository maps, and source-only zip handoffs to support repeatable AI-assisted development cycles.
- Designed SkyWeb/SkyServer boundaries to support future workflow orchestration, alert automation, ingestion hardening, and external notification delivery.
