# SkyWeb Screenshots

Place portfolio screenshots for SkyWeb Analytics in this folder.

Canonical filenames are defined in:

```text
docs/SkyWeb_Visual_Asset_Manifest.md
```

Capture guidance is defined in:

```text
docs/SkyWeb_Screenshot_Capture_Guide.md
```

Keep this folder free of secrets, local terminal captures, database connection details, browser profile details, and temporary throwaway images.
